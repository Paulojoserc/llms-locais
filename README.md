# 🧠 LLMs Locais

Estudos sobre Inteligência Artificial Local utilizando Ollama, Odysseus e modelos de linguagem executando 100% offline.

Este repositório documenta minha jornada aprendendo:

- LLMs Locais
- Benchmarks
- Agentes de IA
- Tool Calling
- Automação
- Infraestrutura
- IA aplicada ao ambiente corporativo

---

# Objetivo

Meu objetivo é entender quais modelos oferecem o melhor equilíbrio entre:

- qualidade
- velocidade
- consumo de memória
- capacidade de programação
- aplicação em infraestrutura de TI

Todos os testes são realizados utilizando exatamente o mesmo hardware e os mesmos prompts.
